\section{MT3DMS Exercise 1, Part 1: Single-Species 2-D Transport Simulation}
The main purpose of this first exercise is to learn the use of MODFLOW and MT3DMS 
for the analysis of a 2-D contaminant transport problem under both 
natural-gradient and under manipulated hydraulic conditions. 
The effect of different transport solution techniques on 
plume migration will be illustrated.

\subsection{Background: Groundwater Flow}
A simple synthetic aquifer system is considered.
A sketch of the problem is shown in Figure \ref{fig:single_spec_fig1}. 
The aquifer is unconfined and drained by a river. We consider that the flow is mainly perpendicular
to the river. In the numerical model a fixed head boundary can be assumed 
for the western boundary of the domain. The southern and northern boundaries 
can for simplicity be considered as no-flow boundaries assuming that the flow is mainly 
oriented from the West towards the East. 
The model domain has an extension of 250 m in East-West direction and 120 m in North-South direction.
A hydraulic head of 11 m represents the situation at the western boundary. 
A fixed hydraulic head of 10.0 m represents the river water level, assuming that 
the aquifer and the river have a good hydraulic connection. 
A uniform background horizontal hydraulic conductivity of 10 m/d is assumed. 
A zone with a a slightly higher hydraulic conductivity prevails in a zone of old 
river channel deposits. A drinking water extraction 
well is located in the South-East corner of the model domain.

and southern model boundaries are implicitly defined as zero mass flux conditions. 
All relevant physical transport parameters are listed in Table \ref{single_spec_table_1}.
They are assumed to be uniform throughout the whole model.
In the exercise we will use the model to determine the maximum allowable 
pumping rate in the extraction well such that the contaminant 
concentrations in that well will remain below 10 \micro g/L.
We will test various model scenarios and assess the affect of the model discretisation
on the model results and of the selection of the solution technique for advectice transport.

\subsection{Running the flow model}

You have now completed to set up the flow model and you are ready to run MODFLOW.
This step will provide MT3DMS with the flow field that is used to compute 
advective-dispersive transport.

\begin{itemize}

\item Select {\color{blue}Model Parameters} $\rightarrow$ {\color{blue}2.Flow} $\rightarrow$ and press 
the {\color{blue}Write Button}.

\item Select {\color{blue}Model Parameters} $\rightarrow$ {\color{blue}2.Flow} $\rightarrow$ and press
the {\color{blue}Run Button}.

\item When the calculation has finished, a window appears, which shows the last lines of the 
main MODFLOW output file. Verify that the last line shows the total simulation time (here 800 days).  
If this is not the case then there was an error in the run and the model input needs to be reassessed and debugged.


\end{itemize}

\subsection{Setting up the transport model}

\begin{itemize}

\item Set the boundary conditions for concentrations by selecting 
{\color{blue}Spatial Variables} $\rightarrow$ {\color{blue}MT3DMS} 
$\rightarrow$ {\color{blue}btn.12 Boundary Conditions} (this is the ICBUND value). 
The value must be equal to 1 (= active cell) throughout the domain. In addition 
a {\color{blue}Zone} must be created (a polygon) to approximately replicate 
the source and the value for this {\color{blue}Zone} must be set to -1 
to specify a constant concentration boundary condition.

\item Define the initial concentrations by selecting 
{\color{blue}Spatial Variables} $\rightarrow$ {\color{blue}MT3DMS} 
$\rightarrow$ {\color{blue}btn.13 Concentrations}. 
Use the same coordinates as defined previously to allocate 
an initial concentration of 10 mg/L to this {\color{blue}Zone}.

\item Define the advection package by selecting 
{\color{blue}Model Parameters} $\rightarrow$ {\color{blue}3.Transport} $\rightarrow$ 
{\color{blue}Model Parameters}. In the dialog box choose {\color{blue}ADV} and  
{\color{blue}adv.1} then {\color{blue}MIXELM: Method}. 
Select {\color{blue}TVD} for the initial transport model runs.

\item Define the dispersivity to be used by selecting (in the same dialog), 
the {\color{blue}DSP} $\rightarrow$ {\color{blue}dsp.2} option and 
enter a value of 1 for the longitudinal 
dispersivity. Enter also 0.1 for TRPT under {\color{blue}DSP} $\rightarrow$ {\color{blue}dsp.3}. 

\item Run the transport model by selecting 
{\color{blue}Model Parameters} $\rightarrow$ 
{\color{blue}3.Transport} $\rightarrow$ and pressing the {\color{blue}Write Button}. 
This will write all required {\color{blue}MT3DMS} 
input files. Then press the {\color{blue}Run Button} to start the transport simulation.

\item A MsDos window will appear and you will see rapidly streaming numbers 
until the simulation is finished. After the end of the simulation
you should see the message $END$ $OF$ $MODEL$ $OUTPUT$. 

\end{itemize}

\subsection{Breakthrough curves and contour plots}

To visualise the breakthrough curve for the contaminant at a specific
location proceed as follows:

\begin{itemize}

\item Visualize the concentration contours under
{\color{blue}Results} $\rightarrow$ {\color{blue}3.Transport} $\rightarrow$ {\color{blue}Tracer}. 
You can change the {\color{blue}Time Step} to visualise the results for other simulation times. 
Also, after selecting a time it is possible to use the mouse wheel to changes times.

\item Define an observation well by going to 
{\color{blue}Spatial Variables} $\rightarrow$ {\color{blue}OBSERVATION} 
$\rightarrow$ {\color{blue}obs.1 Observations } and 
use the {\color{blue}Zone Tool} to draw a point at the desired location.

\item Visualise the breakthrough curve by selecting 
{\color{blue}Results} $\rightarrow$ {\color{blue}5.Observation} $\rightarrow$ {\color{blue}Breakthrough}, 
then select the name of the {\color{blue}Observation} point you just created. 
A dialog will appear. Select {\color{blue}Value} and the breakthrough curve for this 
location will appear. The data used for the curve can be exported and used in a spreadsheet
or other postprocessing tools.

\end{itemize}

\item Rewrite the {\color{blue}MODFLOW} input files (to do so press the {\color{blue}Write Button}

\item To visualise the simulated concentrations at the well an observation point must be
set at the location of the extraction well.

\item Run the model multiple times to determine the maximum pumping rate for which 
the contaminant concentration remains under 10$\micro$g/L.  

\end{itemize}

\subsection{Effect of Solution Techniques}

Now let us first examine the effect of the employed transport solution technique 
on the accuracy of simulation results.

\begin{itemize}
 
\item Export the breakthrough curve obtained at the observation point as obtained when using
the TVD advection option to an ASCII file. 

\item Then change the advection solution option to Upstream Finite-Difference, rerun the model 
and also to MOC, respectively. Export the calculated concentrations to new ASCII files. 
Note that dispersion and sink/source terms are always solved using the standard 
Finite-Difference method and the GCG implicit matrix solver.
Use Excel or other tools to graph the three breakthrough curves.

\end{itemize} 

The TVD and MOC solutions are nearly identical, as both solutions are only 
minimally affected by numerical dispersion. The Upstream Finite-Difference solution, 
on the other hand, shows a significant amount of numerical dispersion.
In addition, check the mass balance errors for each solution. TVD and FD should have
close to zero mass balance errors, while MOC has a large mass balance error at the
beginning. However it should diminish quickly, which is typical for 
particle-tracking based solution techniques.

\subsection{Varying the grid resolution}

With iPHT3D it is possible to easily vary the model discretisation
without the need to change any other model parameters.
To test this effect on the breakthrough curve otained with the Finite-Difference solution technique, 
go to {\color{blue}Model Parameters} $\rightarrow$ {\color{blue}1.Model} 
$\rightarrow$ {\color{blue}Domain} and change the grid cell size.	 

\subsection{Simulating simple contamination remediation methods}
%	 
Go back to the case in which no drinking water extraction takes place and use the model to simulate some basic contaminant remediation and/or attenuation scenarios.
\begin{enumerate}
	\item \textbf{Pump-and-treat}: Illustrate how the contaminant plume could be
      intersected by adding a pump and treat system at >= 80 m downstream of the source zone. Determine the
      pumping rate that would be required to prevent contaminant discharge to the river. Consider 
	  installing multiple wells.
	  %
	\item \textbf{Encapsulate the source zone with a low-permeability barrier}: Illustrate
	  the impact a physical barrier constructed of low-permeability material downstream of the contaminant
	  plume source and/or encapsulate the source zone with low permeability material. Illustrate the impact
	  on both the groundwater flow and on contaminant transport behavior. Experiment with the design of 
	  the barrier and create some figures that document the efficiency of your designs on preventing, 
	  delaying or minimizing contaminant discharge to the river.
	  %	  
	\item \textbf{Natural attenuation scenario}: Add a first-order biodegradation reaction as a chemical
	  reaction into the contaminant transport simulation. Determine the first-order reaction rate constant
	  (and the corresponding half-life) that is required such that the contaminant concentration $C_{dis}$ in
	  the groundwater discharging to the river does not exceed 0.001 $C_{source}$ (ie the concentration you
	  defined at the contaminant source).

\end{enumerate}
